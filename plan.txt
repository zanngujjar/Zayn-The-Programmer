# Blog Section Implementation Plan
# We will call the blog the How to Section on our page as seen in the How to tab in the header
## Overview
Create a comprehensive blog section for the portfolio website with main page, search functionality, and individual blog posts with thumbnails and ad space. Blog posts will use JSON format similar to existing components.

## Components to Create

### 1. Blog Main Page
- Grid layout displaying blog post cards
- Each card contains:
  - Thumbnail image
  - Blog title
  - Excerpt/preview text
  - Publication date
  - Read time estimate
  - Tags/categories
- Pagination for multiple pages of posts
- Recent posts section
- Featured posts section
- Search and filter functionality integrated into main page

### 2. Search and Filter Functionality
- Search bar component with real-time search
- Filter by categories/tags
- Sort options (date, popularity, alphabetical)
- Search results integrated into main page layout
- No results found state
- Keyword-based search with highlighting

### 3. Individual Blog Post Page
- Blog post header with title, date, author
- Featured image
- Blog content area with rich text formatting:
  - Bold text support
  - Italic text support
  - Different heading sizes (H1, H2, H3, etc.)
  - Special `<code>` formatting with syntax highlighting
  - Inline images support
  - Paragraph formatting
- Google Ads integration spots:
  - Header ad space (728x90 or responsive)
  - Mid-content ad space
  - Sidebar ad space (if sidebar layout)
  - Footer ad space
- Social sharing buttons
- Related posts section
- Navigation to previous/next posts
- SEO meta tags with keywords

## Technical Implementation

### File Structure
```
/app/blog
  /page.tsx (Blog main page)
  /[slug]/page.tsx (Individual blog post)
/components/blog
  - blog-card.tsx
  - blog-grid.tsx
  - search-filter.tsx
  - ad-space.tsx
  - blog-content.tsx
  - blog-pagination.tsx
/lib
  - blog-posts.ts (JSON data structure)
```

### Features to Implement
1. Responsive design for all screen sizes
2. SEO optimization with dynamic meta tags and keywords
3. Image optimization and lazy loading
4. Google Ads integration
5. Social media sharing
6. Search functionality with real-time filtering
7. Tag/category system with filtering
8. Reading progress indicator
9. Dark/light mode compatibility
10. Rich text content rendering with custom formatting

### Google Ads Integration
- Create AdSpace component for different ad sizes
- Implement ad placement strategically:
  - Above the fold on blog main page
  - Within blog content (after 2-3 paragraphs)
  - Sidebar ads on desktop
  - Between related posts
- Ensure ads are responsive and mobile-friendly
- Add proper ad labels for compliance

### Content Management
- JSON-based blog storage in TypeScript file
- Rich text content with custom formatting support
- Image storage in public folder
- SEO metadata fields for each post
- Search indexing with keyword matching

## Implementation Priority
